# Employee-Tracker


## Description

This is a command-line application built to help manage a company's employee database, using Node.js, Inquirer, and My SQL.



### Languages use

- Node.js
- Mysql 
- JavaScript 
- MySQL

you us it threw the terminal this is your option 
  * view all departments 
  * view all roles 
  * view all employees

  * add a role
